## 🙋‍♂️ About Me

I'm a **Data Engineer** with **5+ years of experience** designing and building scalable data pipelines, cloud data warehouses, and modern data architectures. I'm passionate about transforming raw data into actionable insights and helping organizations make data-driven decisions.

- 🔭 Currently working with **Databricks**, **Snowflake**, and **dbt** to build enterprise-grade data platforms
- 🌱 Passionate about **Delta Lake**, **Lakehouse Architecture**, and **Data Mesh** principles
- 💡 Love solving complex data challenges — ETL/ELT pipelines, dimensional modeling, and data quality
- 🏆 5+ years of hands-on experience with big data technologies and cloud platforms
- ⚡ Fun fact: I believe clean, well-documented data pipelines are an art form!
